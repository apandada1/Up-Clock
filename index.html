UP Clock

A Clock that was carefully built to be pretty and smart.

Design

UP Clock was inspired by the Ubuntu Touch Lockscreen.
All the design was thought on the user experience, every detail !


Customizable

The configuration is really simple, you can
change the color, set alarm, change the clock type, tranparency, and more...

Download

You can download UP Clock from the PPA in Launchpad or even get the fresh news from GitHub !


If you are using Ubuntu, just open a terminal and enter these commands: 

 sudo add-apt-repository ppa:apandada1/up-clock

 sudo apt-get update

 sudo apt-get install up-clock
